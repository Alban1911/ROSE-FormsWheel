# ROSE-FormsWheel

Custom chroma wheel plugin for Pengu Loader with asset-based buttons.

## Description

ROSE-FormsWheel provides a custom chroma wheel for specific skins (like Sahn Uzal Mordekaiser) with custom button assets. This plugin is adapted from ROSE-ChromaWheel and uses custom image assets instead of the default League of Legends chroma button styling.

## Features

- Custom hover button assets (`hol-button.png`, `hol-button-hover.png`)
- Custom wheel button assets (e.g., `uzal_buttons/*.png`)
- Supports skins with Forms (like Sahn Uzal Mordekaiser)
- Full integration with ROSE backend via WebSocket bridge

## Installation

This plugin is part of the ROSE project and is managed as a git submodule. To use it, ensure it's properly initialized:

```bash
git submodule update --init --recursive
```
